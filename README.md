### Sorting
- Bubble
  - O(n^2)
  - perform badly -> good for detect if a list is already sorted. (O(n))

- Insertion
  - O(n^2)
  - More efficient tha bubble / selection
  - Stable
  - Can sort a list as it receives it

- Selection
  - O(n^2)
